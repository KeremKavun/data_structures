# Data Structures (C)

A collection of data structures implementations i implemented through data structres course.

## Overview

This repository contains modular implementations of arrays, lists, trees, queues, stacks, graphs, and related utilities. Each module lives in its own directory and includes source, headers, and tests where applicable.

## Project Structure

The library is organized into specialized modules:

- `include/`: Public API headers.
- `src/`: Core implementation logic.
- `tests/`: Comprehensive unit and integration tests.

### Modules & API Reference

- **[Arrays](@ref ARRAYS)**: Contiguous memory wrappers and dynamic vectors (DynArrays).
- **[Linked Lists](@ref CircularLinkedList)**: Intrusive circular and linear linked structures.
- **[Trees](@ref TREES)**: Binary Trees, Heaps, AVL, and BST implementations.
- **[Queue](@ref QUEUE)**: FIFO containers (Linked and Array-based).
- **[Stack](@ref STACK)**: LIFO containers with stack-frame safety.
- **[Graphs](@ref GRAPHS)**: Adjacency-list representations for directed/unweighted graphs.
- **[HashStructs](@ref HASHS)**: High-performance lookup tables (HashSet s in progress).
- **[Utilities](@ref UTILS)**: Logging (@ref DEBUG), @ref OBJ_CONCEPT, and @ref ALLOC_CONCEPT.
